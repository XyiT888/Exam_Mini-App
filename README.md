## 1.4 结果界面

退出考试-->开始界面

**遇到的问题：**

在点击退出考试按钮时，并不能退出到开始考试界面，即点击该按钮没有反应。因为考试须知界面的退出按钮和结果界面的退出按钮执行的功能是一样的，所以我将两个按钮设置了相同的ID，但在HTML中，ID应该是唯一的，重复的ID会导致JavaScript只能获取到第一个匹配的元素。重新设置结果界面的退出按钮id即可解决。
